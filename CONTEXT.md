## 戦闘メカニクス（実データから解明済み）

### 三すくみターゲティング（最重要）
各兵種は三すくみの有利な相手を**直接攻撃**する（前線システムではない）：
- 盾 → 敵の槍を攻撃（密集戦陣: +10%）
- 槍 → 敵の弓を攻撃（突撃: +10%）
- 弓 → 敵の盾を攻撃（遠距離打撃: +10%）
優先ターゲットが全滅したらフォールバック（盾→槍→弓の順）

**結果**: 盾が最大損失、弓が次、槍はほぼ無傷（バトルレポートで確認済み）

### ダメージ計算式
```
kills = C × sqrt(自軍兵数) × heroStatRatio × SkillMod × TypeBonus × GearMod × GemMod × jitter
```
- C = 20（キャリブレーション済み）
- heroStatRatio = 攻撃側(ATK×Leth) / 防御側(DEF×HP)
- SkillMod = (damageUp × oppDefenseDown) / (defenseUp × oppDamageDown)
- TypeBonus = 1.10（三すくみ有利時）
- GearMod = 領主装備+英雄装備+ペットの差分
- GemMod = 宝石の兵種別差分
- jitter = 0.92〜1.08

### スキル発動ルール
- リーダー3体: S1/S2/S3全て発動
- ライダー: S1のみ、上位4枠、重複なし、ステータスは無関係
- always: 毎ターン発動
- prob: 確率で毎ターン判定
- periodic: N回/Nターン毎に1回（period値を使用）
- target限定スキル: 兵比で按分（例: 盾限定+100%で兵比5:0:5 → 実効+50%）

### 集結の仕組み
- 兵種レーン制の兵士プール方式（英雄vsマッチアップではない）
- 英雄はグローバルバフ提供者
- リーダーのベースステータスは自兵種にのみ適用（バトルレポートで確認）
- ただしスキル効果は全兵種にグローバル適用

### 負傷/死亡
- 攻撃敗北: 27%死亡 / 13%重傷 / 60%軽傷
- 防御敗北: 0%死亡 / 35%重傷 / 65%軽傷
- 勝利側: 4%死亡 / 46%重傷 / 50%軽傷
