## 概要

- **部位数**: 4部位（共通装備） + 1部位（専用装備、SSR英雄のみ）
- **品質（レアリティ）**: 6段階（グレー → グリーン → ブルー → パープル → ゴールド(神話) → レッド(レジェンド)）
- **最大強化レベル**: Lv.100
- **解放条件**: 大溶鉱炉レベル15
- **精錬鍛造（Mastery Forging）解放**: 大溶鉱炉レベル20 + ゴールド装備Lv.20
- **昇格（Ascension）解放**: 神話装備Lv.100 + 精錬Lv.10

## 強化システム

### 1. 通常強化（Enhancement）

- 強化経験値パーツまたは他の英雄装備を消費してレベルアップ
- 最大レベル: Lv.100（神話/レジェンド）、Lv.80（エピック、大溶鉱炉ランク依存）
- 合計必要経験値（Lv.100まで）: **73,320 XP**
- レベルアップごとに英雄ステータス（ATK/DEF/HP）とコマンド値が増加

#### エピック装備の強化上限（大溶鉱炉ランク依存）

| 大溶鉱炉ランク | 強化上限レベル |
|-------------|-----------|
| 25未満 | Lv.69 |
| 25以上26未満 | Lv.74 |
| 26以上27未満 | Lv.79 |
| 27以上 | Lv.80 |

### 2. 精錬鍛造（Mastery Forging）

- ゴールド（神話）品質の装備のみ対象
- 解放条件: 大溶鉱炉Lv.20 + 装備強化Lv.20
- 最大精錬レベル: Lv.20
- 精錬によりステータスに追加%ボーナスが加算（レベルごとに約10%ずつ増加とされる）

### 3. 昇格（Ascension）: 神話 → レジェンド

- 条件: 神話装備Lv.100 + 精錬Lv.10
- 素材: 神話英雄装備 x2
- 昇格後も最大強化レベルはLv.100

## 専用装備（Exclusive Gear）

SSR（ゴールド品質）英雄のみが装備可能な5番目の装備スロット。
英雄を★1に覚醒させると解放される。

### 専用装備の特徴

- **ステータス**: 英雄攻撃力 + 英雄防御力 + 英雄HP + 殺傷力コマンド + HPコマンド
- **専用スキル**: 2種類（探検スキル + 遠征スキル）を追加
- **強化**: 専用ウィジェット（Widget）を消費してレベルアップ（最大Lv.10）

### 専用装備の強化コスト

| レベル | 必要ウィジェット数 |
|--------|--------------|
| 解放 | 5 |
| Lv.1 | 10 |
| Lv.2 | 15 |
| Lv.3 | 20 |
| Lv.4 | 25 |
| Lv.5 | 30 |
| Lv.6 | 35 |
| Lv.7 | 40 |
| Lv.8 | 45 |
| Lv.9 | 50 |
| Lv.10 | 50 |
| **合計** | **325** |

### 専用装備のステータス範囲（世代別パワークリープ）

| 世代 | 兵種殺傷力/HP | 例 |
|------|-------------|------|
| 初期～第4世代 | +50% ~ +232% | Jasmine, Gina, Molly等 |
| 第5～第8世代 | +232% ~ +400% | - |
| 第9世代以降 | +447.5% ~ +490% | Eliose, Hank等 |

## 強化優先度ガイド

### 英雄装備の強化推奨順

1. **盾兵のグローブ・ベルト**（最優先）→ 盾兵HPコマンドUP → 壁役の生存力向上
2. **弓兵のゴーグル・ブーツ** → 弓兵殺傷力コマンドUP → DPSの火力向上
3. **槍兵のゴーグル・ブーツ** → 槍兵殺傷力コマンドUP
4. **盾兵のゴーグル・ブーツ**（後回し可）
5. **弓兵・槍兵のグローブ・ベルト**（優先度低）

## 品質別ステータス概算値（未検証・要ゲーム内確認）

> !! 以下は各種ガイドの記述から推定した概算構造です。
> 正確な数値はゲーム内スクリーンショットによる検証が必要です。

英雄装備のステータスは以下の要素で決まる:
1. **品質（レアリティ）** → 基礎ステータス上限を決定
2. **強化レベル（Lv.0～100）** → レベルごとにステータスが線形増加
3. **精錬レベル（Mastery Lv.0～20）** → 基礎ステータスに対する%加算ボーナス
4. **賦能（Empowerment）** → レジェンド装備のマイルストーンボーナス

### 品質別の相対的な強さ

| 品質 | 相対パワー（概算） |
|------|--------------|
| グレー | 基準値（強化不可） |
| グリーン | グレーの約3倍 |
| ブルー | グレーの約6倍 |
| パープル | グレーの約15倍 |
| ゴールド（神話） | 最高品質（精錬込みで大幅増） |
| レジェンド | 神話を超える最終段階（賦能込み） |

## 領主装備（チーフギア）との違い

| 項目 | 英雄装備（Hero Gear） | 領主装備（Chief Gear） |
|------|---------------------|---------------------|
| 装備対象 | 各英雄個別 | 領主（全英雄に間接影響） |
| 部位数 | 4 + 1（専用） | 6 |
| ステータス種類 | 英雄ATK/DEF/HP + 兵種コマンド | ATK%/DEF% + 出撃部隊容量 |
| ティア表記 | 品質色（グレー～レッド） | グッド～神話T4 |
| 強化方式 | XPレベル制（Lv.1-100） | ティア昇格 + ★精錬 |
| 追加システム | 精錬鍛造、昇格、賦能 | 陶冶（チャーム） |
